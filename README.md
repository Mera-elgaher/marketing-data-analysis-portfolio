# Marketing & Services Data Analysis Portfolio

This repository showcases my experience in analyzing marketing and service performance data,
focusing on extracting insights, evaluating ad effectiveness, and providing data-driven recommendations.

## Project Overview
The project focuses on analyzing marketing data for service-based businesses such as:
- Mobile repair services
- Home services

The goal is to understand:
- Which ads perform best
- Which services generate higher demand
- Which locations have higher conversion rates

## Analysis Scope
- Ad performance evaluation (Clicks, Conversions, Cost, ROI)
- Service demand analysis
- Location-based insights
- Key performance metrics (CTR, Conversion Rate)

## Tools Used
- Excel / Google Sheets
- Data Visualization
- Analytical Reporting

## Deliverables
- Cleaned and structured datasets
- Visual insights (charts & summaries)
- Business recommendations based on data patterns

## Note
All data used in analysis projects are either anonymized or sample-based to maintain data privacy.
